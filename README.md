# 구현 기능

### 강의 구현 기능

- Work, Travel 탭
- 입력시 Todo 추가 기능
  - Work에서 쓰면 Work리스트에 Travel에 쓰면 Travel리스트에 추가
- Todo 삭제 기능
- 데이터 AsyncStorage에 저장

### 추가 구현 기능

- 앱 재실행시, 마지막 상태의 Work 또는 Travel 기억
- Todo 완료 기능
- Todo 수정 기능
